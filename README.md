# 🏡 AwayHome

AwayHome is a modern real estate web application built with Next.js and Tailwind CSS. It focuses on delivering a clean, responsive, and visually engaging experience for showcasing premium properties.

---

## ✨ Features

- 🧭 Minimal and modern navigation
- 🏠 Beautiful hero section with featured property
- 📩 Newsletter subscription UI
- 🌙 Premium dark footer with gradient effects
- ⚡ Fast and responsive design
- 🎨 Clean UI built with Tailwind CSS

---

## 🚀 Tech Stack

- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Language:** TypeScript
